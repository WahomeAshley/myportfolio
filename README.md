# Skatejump
# Jvascript + three.js

Skate Runner 3D is a simple endless runner-style skateboard game built using JavaScript and Three.js. The player controls a skateboarder who automatically moves forward along a street, dodging and jumping over obstacles to survive as long as possible.
#Game Concept
The game takes place on an endless road where the skateboarder is constantly moving forward. The main goal is to avoid crashing into street obstacles such as cans, boxes, and cones by jumping or moving sideways. The longer you survive, the higher your score.
#Controls
Space / Up Arrow – Jump
Left / Right Arrow (optional) – Move between lanes
